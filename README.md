¡Bienvenido/a al repositorio **Fundamentos de Python para Ciberseguridad**! Este espacio está diseñado para recopilar scripts, herramientas y prácticas orientadas a la automatización de tareas de seguridad, auditorías ofensivas (Pentesting) y defensa activa utilizando Python.

---

## 🎯 Propósito del Proyecto

El objetivo de este repositorio es servir como una guía práctica y estructurada para aprender a utilizar Python como el lenguaje de script principal en entornos de ciberseguridad. El contenido abarca desde los fundamentos sintácticos del lenguaje hasta el desarrollo de herramientas de red, web y criptografía.

---

## 📂 Contenido del Repositorio

De acuerdo con la estructura actual del proyecto (ver `image_8723e8.png`), los módulos se dividen de la siguiente manera:

* **`01 · Fundamentos de Python para Ciberseguridad`**: Introducción a la sintaxis de Python, variables, estructuras de control, funciones y conceptos esenciales aplicados al ámbito informático.
* **`02 · Manejo de Archivos, Expresiones Regulares y Automatización`**: Procesamiento de logs, lectura/escritura de archivos del sistema, automatización de tareas y uso de *Regex* para buscar patrones (como IPs, correos o credenciales).
* **`03 · Sockets y Reconocimiento de Red`**: Programación con la librería `socket`, creación de escáneres de puertos, *banner grabbing* y scripts de reconocimiento pasivo y activo en redes.
* **`04 · Peticiones HTTP, Web Scraping y Auditoría Web`**: Interactuación con aplicaciones web, uso de librerías como `requests` y `BeautifulSoup` para auditorías automatizadas, fuzzing de directorios y extracción de información (OSINT).
* **`05 · Hashing y Cracking de Contraseñas`**: Uso de funciones criptográficas y hashes (`hashlib`), generación de payloads básicos y desarrollo de scripts para ataques de fuerza bruta o diccionarios (*Password Cracking*).
